**CASHKING DOWNLOADER (CKD)** is a fast, reliable, and universal **video/audio downloader API** built with **FastAPI** and **yt-dlp**.  
It allows you to download videos from **TikTok, YouTube, Facebook, Telegram, and more**, directly via a shareable link ‚Äî **no watermark included**.  

---

## üöÄ Features

- Download videos or audio from any supported platform  
- Automatically detects platform and video metadata  
- Returns direct download URLs  
- Simple JSON-based API  
- Ready for frontend integration  
- Handles TikTok short links (`vm.tiktok.com`) automatically  
- Works locally or hosted online
